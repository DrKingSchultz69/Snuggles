# Snuggles – Modern Fashion E-Commerce Platform (PRD)

## Vision
Create a luxury-minimal fashion e-commerce experience that feels high-end, editorial, and confident. The design language should be bold typography, lots of whitespace, neutral palette, immersive photography, and subtle micro-animations.

The website should feel:
- Premium
- Editorial
- Clean
- Timeless
- International

## 1. 🎯 Objectives
- Establish Snuggles as a premium contemporary fashion brand
- Deliver a seamless mobile-first shopping experience
- Achieve fast load speeds (<2.5s LCP)
- Increase product discovery via clean navigation
- Create a strong visual brand identity

## 2. 🧑‍🤝‍🧑 Target Audience
- Age: 18–35
- Urban, fashion-conscious
- Instagram-native shoppers
- Prefers minimal aesthetics
- Mid-premium price segment

## 3. 🖥️ Core Pages & Structure

### 3.1 Homepage
**Layout Sections:**
- **Full-Width Hero**
  - Large editorial image (placeholder)
  - Minimal overlay text: SNUGGLES | NEW COLLECTION
  - CTA: SHOP NOW
- **Category Blocks (Grid 2x2)**
  - Women
  - Men
  - Accessories
  - New In
- **Editorial Campaign Section**
  - Full width image
  - Large serif typography headline
  - CTA: DISCOVER
- **Trending Now**
  - 4–8 product grid
  - Hover effect: image swap
- **Newsletter Section**
  - Minimal email input
  - Black button

### 3.2 Navigation
**Top Bar**
- Left: Menu (☰)
- Center: SNUGGLES logo (text-based)
- Right: Search | Account | Bag

**Menu Drawer**
- Women
- Men
- New
- Collections
- Editorial
- Sale

### 3.3 Product Listing Page (PLP)
**Layout:**
- 3–4 column grid (desktop)

**Each product card:**
- Large image

**Filters (minimal left sidebar or slide-out):**

- Color

- Left: Large image gallery

**Sections:**

### 3.5 Cart & Checkout

## 4. 🎨 Design System

### Typography
- **Primary:** Modern Sans Serif (e.g., Helvetica/Inter style)
- **Secondary:** Elegant Serif (editorial headlines)
- **Style:** ALL CAPS for navigation, Large spaced typography, Generous whitespace

### Color Palette
- Black: #000000
- White: #FFFFFF
- Soft Beige: #F5F5F5
- Warm Grey: #DCDCDC
- No bright accent colors.

### Imagery Style
- High contrast
- Neutral tones
- Editorial poses
- Natural lighting
- Clean backgrounds
